# SecureScan – Web Vulnerability Scanner

A lightweight Python‑based web vulnerability scanner inspired by OWASP practices.  
Checks HTTPS, security headers, and open ports, with a Flask dashboard and CSV export.

## Features

- Detects missing HTTPS vs HTTP.
- Checks for missing security headers (`X-Content-Type-Options`, `X-Frame-Options`, `CSP`, `HSTS`).
- Scans ports 80, 443, 8080.
- Exports results to CSV.
- Clean Bootstrap 5 dashboard.
